# Personal Website 

Overview
This project is a personal website built with Next.js and deployed on Vercel.  
The goal of the site is to provide a clean, modern, and responsive platform to showcase personal information, projects, and ways to connect with the developer.

Purpose of the Website
The website is designed to act as a digital portfolio and personal hub. Visitors can quickly learn about the developer, explore past and ongoing projects, and find direct links for communication.  
The emphasis is on simplicity, speed, and accessibility across all devices.

Key Features
- Modern and minimalistic design  
- Fast loading times thanks to Vercel hosting  
- Fully responsive layout for desktop, tablet, and mobile  
- Easy scalability to add new pages or features  

Running Locally
To run the project on your local machine:

1. Clone the repository:
   `bash
   git clone <https://alhadimohammed.github.io/-my-Blog/>
   `
2. Navigate into the project folder:
   `bash
   cd project-name
   `
3. Install dependencies:
   `bash
   npm install
   `
4. Start the development server:
   `bash
   npm run dev
   `

Open your browser at http://localhost:3000 to preview the site.

Deployment
The project is deployed using Vercel, a powerful platform for hosting modern web applications.  
Every time changes are pushed to the repository, Vercel automatically builds and deploys a new version of the site.

Technologies Used
- Next.js: Framework built on React for modern web development  
- React: Library for building user interfaces  
- Vercel: Hosting and deployment platform  

Detailed Explanation of the Website
The website consists of several sections:  

- Home Page: Introduces the developer with a short bio and a welcoming design.  
- Projects Section: Displays past and current projects, highlighting skills and achievements.  
- Contact Section: Provides links to email and social media accounts for easy communication.  

From a technical perspective, the site leverages Next.js features such as Static Site Generation (SSG) and Server-Side Rendering (SSR) to ensure high performance and smooth user experience.  
Deployment on Vercel allows for instant updates, version control integration, and global CDN distribution, ensuring the site is fast and reliable worldwide.
